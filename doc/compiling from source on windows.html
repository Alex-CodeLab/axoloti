Compiling from source on Windows

Get the gnu arm compiler, build tools and dfu-util

Automated by running the shell script "platform_win\get_dependencies.sh" from the MinGW shell. If you don't want to install MinGW, open the shell script in a text editor and do everything by hand...

Install Java

To compile from source, you need Java JDK, and Ant. I recommend downloading the Netbeans bundle with Java SE from here: https://netbeans.org/downloads

Compiling the Axoloti GUI

Open the project in NetBeans, go to the "Run" menu, and select "Build Project".

Or alternatively, launch the batch file "compile_gui.bat" in the "platform_win" directory.
